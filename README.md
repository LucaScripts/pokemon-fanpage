## Sobre o Projeto

Uma fanpage pessoal construída com HTML, CSS e JavaScript puro, inspirada na estética retrô dos jogos clássicos de Pokémon. Sprites animados, trilha sonora nostálgica e um Pikachu interativo fazem parte da experiência.

---

## Funcionalidades

<img src="https://raw.githubusercontent.com/PokeAPI/sprites/master/sprites/pokemon/versions/generation-v/black-white/animated/shiny/25.gif" align="right" width="90"/>

- **Tema claro / escuro** com ícones adaptáveis
- **Playlist de músicas** retrô com troca de faixa no mute/unmute
- **Badge de música** animado com nome da faixa atual
- **Cards de navegação** com sprite estático e GIF animado no hover
- **Pikachu interativo** — clique para ouvir o grito e ver a versão shiny
- **Hover sonoro** no Pikachu (blip chiptune retrô)
- **Cursor personalizado** de Poké Ball
- **Painéis laterais** com conteúdo: Sobre, Links, Pokédex, FAQ, Contato e Extras
- **Ícones de itens** da PokéAPI substituindo emojis
- **Redes sociais** no rodapé
- **Layout responsivo** para mobile

## Tecnologias

- **HTML5** — estrutura semântica
- **CSS3** — variáveis, `clamp()`, flexbox, grid, animações
- **JavaScript** — Web Audio API (SFX), HTML5 Audio (playlist), PokeAPI sprites
- **[PokeAPI Sprites](https://github.com/PokeAPI/sprites)** — sprites e ícones de itens (CC0)
- **[Press Start 2P](https://fonts.google.com/specimen/Press+Start+2P)** — fonte retrô pixel
- **[Nunito](https://fonts.google.com/specimen/Nunito)** — fonte de texto

## Como Usar

```bash
# Clone o repositório
git clone https://github.com/LucaScripts/pokemon-fanpage.git

# Abra o arquivo no navegador
pokemon-fanpage.html
```

> Não requer instalação nem servidor — é HTML puro. Basta abrir no navegador.
